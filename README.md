# GlsUni_mob_prototype

A responsive mobile-first prototype for the GLS University Student Portal.

This project uses HTML, Tailwind CSS, and Vite.

## Setup and Development

1. Run `npm install` to install dependencies.
2. Run `npm run dev` to start the development server.

## Deployment

This project is configured to be deployed out-of-the-box on Vercel. Simply import the repository into Vercel and it will automatically detect the Vite build settings.
